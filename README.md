# Franka LLM

Franka robot manipulation system with LLM-based planning and vision-based object detection.

## Branch Structure

- **`main`** - Stable baseline for milestone releases (current branch)
- **`dev`** - Active development branch
- **Milestone branches** - Archived versions of important demos/milestones

### Milestone Branches

- `demo-roel-feb11-2026` - Demo shown to Roel on Feb 11, 2026

## Development Workflow

1. Active development happens on `dev` branch
2. After reaching a milestone, a new branch is created to preserve that state
3. Main branch serves as the stable baseline

For detailed setup and usage instructions, see [QUICKSTART.md](QUICKSTART.md).
